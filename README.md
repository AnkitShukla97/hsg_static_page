# Assignment Summary

This project was built as a React + Vite static web page based on a provided PSD design. The objective was to convert the design into a responsive website using React, with custom CSS and no Tailwind.

## Assignment Objective

Convert the given PSD into a static web page using React with responsive screens.

## Requirements Covered

The page includes the following sections:

1. Header with mega menu
2. Slider banner
3. Content section with images and icons
4. Center slider section
5. Footer with plain text

## Tech Stack

- React
- Vite
- Custom CSS
- Responsive design

## Notes

The implementation focuses on translating the visual design into a clean and responsive frontend experience while maintaining a structure suitable for interview discussion.

## Project Purpose

This assignment demonstrates frontend development skills in:

- building UI components in React
- creating responsive layouts
- styling with custom CSS
- structuring a static page from a design reference
